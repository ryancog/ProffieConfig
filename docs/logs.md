# Log Files

ProffieConfig stores log files that record information, warnings, and errors as it runs.
These can often be helpful when trying to figure out why something isn't working quite right.

If the app crashes, or encounters something problematic, it'll often show a dialog which inclues a "Show Log Folder" button. This'll bring you straight to the folder, but sometimes this doesn't show up.

## Where to find them.

### On Windows:
In File Explorer, navigate to `%LOCALAPPDATA%\ProffieConfig`. You can type or copy/paste that into the path bar at the top to "jump" straight to the folder.

### On macOS:
In Finder, press `Cmd+Shift+G` to "Go To Folder," type in `~/Library/Logs/ProffieConfig`, and press enter to show the log folder.

## What to do with them?

If you're having an issue with the ProffieConfig Launcher (i.e. during the installation/update process), grab the `ProffieConfig Launcher.log` file, otherwise grab the `ProffieConfig.log` file (which might not exist if the app hasn't launched successfully yet). If you're not sure, it doesn't hurt to send both.

Depending on the issue, and if you're savvy, you might be able to read through and figure out what's going on yourself, but chances are it's a problem internally in ProffieConfig or you might just need help to figure it out.

Send me an email or otherwise contact me and include the log file, along with any information about what's going wrong and I can try to help out.
